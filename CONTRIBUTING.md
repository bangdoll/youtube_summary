# 貢獻指南 (Contributing Guide)

感謝您有興趣為 **Second Brain OS (原: Youtube 智慧大腦)** 做出貢獻！

## 👋 如何參與

我們歡迎任何形式的貢獻，包括但不限於：
1.  **回報錯誤 (Bug Reporting)**：請在 Issues 中詳細描述問題的重現步驟。
2.  **功能建議 (Feature Requests)**：如果您有新點子，歡迎提出討論。
3.  **程式碼貢獻 (Pull Requests)**：修復錯誤或實作新功能。

## 🛠 開發流程

1.  **Fork** 此專案。
2.  **Clone** 到您的本機環境：
    ```bash
    git clone https://github.com/YOUR_USERNAME/youtube_summary.git
    ```
3.  **建立分支**：
    ```bash
    git checkout -b feature/AmazingFeature
    ```
4.  **提交變更** (請使用繁體中文 Commit Message)：
    ```bash
    git commit -m "feat: 新增了令人驚艷的功能"
    ```
5.  **Push** 到您的 Fork：
    ```bash
    git push origin feature/AmazingFeature
    ```
6.  **建立 Pull Request**。

## 📝 程式碼風格與規範

- **語言**：所有文件、註釋與介面文字必須使用 **繁體中文 (Traditional Chinese)**。
- **Python**：請遵循 PEP 8 風格。
- **前端**：使用現代化 Vanilla JS/CSS，保持簡潔。

## ⚠️ 注意事項

- 本專案整合了多種 AI 模型 (Gemini, GPT-4o)，請確保您的貢獻不會導致 API 金鑰洩露。
- 測試時請注意 API 配額使用量。
